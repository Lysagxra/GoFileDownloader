# Contributing

Thanks for considering contributing! Contributions are welcome.

## How to Contribute

1. **Fork the repository** and clone it locally
2. **Create a branch** for your changes: `git checkout -b feature/my-feature`
3. **Make your changes** and test them
4. **Commit** with clear messages
5. **Push** to your fork and open a pull request

## Code Style

Run `ruff check .` before committing to ensure code quality.

## Reporting Issues

Open an issue with a clear description of the problem or feature request.

Thanks for contributing!
